**Project Overview**  
Create a backend Python project using agents framework that provides voice agents (11labs) run by manager (hermes)

**Technical Requirements**  
\- Language: Python  
\- Framework: Hermes Agent \- [https://github.com/nousresearch/hermes-agent](https://github.com/nousresearch/hermes-agent) or [https://github.com/aaronjmars/MiroShark](https://github.com/aaronjmars/MiroShark)  
\- [11labs voice agents](https://elevenlabs.io/agents)   
\- [Airtable](https://www.airtable.com/)  
\- LLM Provider: [openrouter](http://openrouter.ai) \+ any free model you want  
\- [apify](https://apify.com/)\- for data scraping (its free)

**Project Scope**  
Develop agents with flow that:

1. fetch debt size/ date / name / phone \# / product / company name  from Airtable / gsheet you can create any data you want as long its logical)  
2. hermes agent to find when to call the debtor based on holiday calendars / weekends / hour of the day  (use apify to fetch holidays)  
3. Hermes agent using a dedicated skill to trigger the voice bot in 11labs through 11labs agent when need to call   
4. debtor (client) can chat with the collection agency about its debt on the whatsapp (you need to define the 11labs agent as well)  
5. make sure the agent doesn't answer anything not related to his job…  
6. implement “A closed learning loop” based on Hermes builtin functionality to see that cal to debtor can be improved by results of collection call  
7. this will connect later with [https://telnyx.com/](https://telnyx.com/) for actual call on the phone with debtor (not for this task)
